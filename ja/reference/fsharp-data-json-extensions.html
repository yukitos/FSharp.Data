F# Data

Extensions

Unsafe extension methods that can be used to work with JsonValue in a less safe, but shorter way. This module also provides the dynamic operator.

Functions and values

Function or valueDescription
( ? ) jsonObject propertyName
Signature:jsonObject:JsonValue -> propertyName:string -> JsonValue
